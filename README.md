## Featured Projects
### [CampusHive - University ERP Platform](https://github.com/samriddhiyadav/campushive-university-erp)
Multi-role ERP for student, faculty, HR, and finance workflows with JWT/SSO authentication and modular backend services.
- Architecture: role-aware API layer + normalized MySQL schema + secure auth boundary.
- Stack: Node.js, Express.js, MySQL, JWT, SSO.
- Engineering focus: access control granularity, query indexing, maintainable service design.

### [SmartEdu - AI Adaptive Learning Platform](https://github.com/samriddhiyadav/smartedu-ai-learning-platform)
Adaptive learning platform with ML-driven personalization delivered under hackathon time constraints.
- Architecture: recommendation engine + learning telemetry + role-based web app.
- Stack: Python, Scikit-learn, Node.js, React.
- Engineering focus: rapid model integration, explainable recommendations, real-time signal loops.

### [TalentIQ - AI Recruitment Intelligence System](https://github.com/samriddhiyadav/talentiq-ai-recruitment-engine)
AI-powered hiring platform for resume ranking, semantic role matching, and interview readiness scoring.
- Architecture: ML inference service + recruiter workflow backend + analytics plane.
- Stack: PHP, Python ML, MySQL.
- Engineering focus: scalable inference path, model lifecycle strategy, multi-role data segregation.

### [JobGenie - AI Voice Interview Simulator](https://github.com/samriddhiyadav/jobgenie-ai-interview-simulator)
Voice-driven interview simulator with generated role-specific questions and score feedback.
- Architecture: conversational API + session store + audio processing pipeline.
- Stack: PHP, MySQL, JavaScript, LLM integration.
- Engineering focus: latency-aware design, persistent feedback state, cloud migration readiness.
